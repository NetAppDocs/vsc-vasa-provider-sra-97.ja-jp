= VSC タスクに必要な権限
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Virtual Storage Console for VMware vSphereのタスクを実行するために必要な権限の組み合わせは、（VSC）とvCenter Server標準の権限のそれぞれに異なります。

VSC タスクに必要な権限については、ネットアップの技術情報アーティクル 1032542 を参照してください。
