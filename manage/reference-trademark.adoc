= 商標です
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


NetApp 、 NetApp のロゴ、および NetApp の商標ページに記載されているマークは、 NetApp, Inc. の商標です。その他の会社名および製品名は、それぞれの所有者の商標である場合があります。

http://www.netapp.com/us/legal/netapptmlist.aspx["ネットアップの商標です"^]
